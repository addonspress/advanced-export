WP Demo Export is very flexible plugin which convenient user to export demo data(posts,page,media and even widget and customizer option ) in zip format from their site.

== Description ==

WP Demo Export is one of the best and powerful demo data exporter plugin. It has number of features which make more manageable and convenient to WordPress user to exact their WordPress site data and again re-use in another website.

Some listed features of WP Demo Export are given below :

* Export widget
* Export option
* Export media,
* Export pages,
* Export post
* Export custom post type
* Export actual media files

== Dashboard Location ==

Dashboard -> Tool -> Demo Export

== Installation ==

There are two ways to install any WP Demo Export Plugin:

1.Upload zip file from Dashboard->Plugin->Add New "upload plugin".
2.Extract WP Demo Export and placed it to the "/wp-content/plugins/" directory.
    - Activate the plugin through the "Plugins" menu in WordPress.
